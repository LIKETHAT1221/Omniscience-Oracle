# Omniscience Oracle 🔮

Advanced sports betting analysis system with technical indicators, reverse line movement detection, and AI-powered recommendations.

## Features

- **Multi-format Parsing**: Handles 4-line, 5-line, and 8-line data blocks
- **Technical Analysis**: MOM-V, MOM-A, RSI, Adaptive Moving Average
- **Reverse Line Movement**: Detects sharp money vs. public money
- **Smart Recommendations**: FADE/BACK/HOLD with confidence scoring
- **Expected Value Calculation**: Kelly criterion stake sizing
